☆はじめに

このたびはデジタル絵本「ねこみみおばけっこ ～春を探そう♪～ 」

をお求めいただきありがとうございます。

デジタル絵本「ねこみみおばけっこ ～春を探そう♪～ 」

は絵・話・構想：雛世ひな　音楽：坂木カエ　による作品です。





☆動作環境

このデジタル絵本をご覧いただくための必要環境は以下のとおりです。

画面解像度  800x600ピクセル以上、High Color以上　(True Color(24ビット)以上を推奨)
PNG・透過PNG形式の画像を表示できる環境　(wwwブラウザでの閲覧を推奨)
CD-ROMディスク対応ドライブ
MIDIを聞ける環境（注：ご使用の環境によっては多少聞こえ方が異なります）

【警告】
本製品はCD-ROMです。音楽が入っていますが音楽CDではありません。
CD-ROMディスク対応ドライブ以外での再生は絶対にしないでください。

動作確認はWindowsXP Professional/Internet Explorer 6.0上でおこなっています。
MIDIはMicrosoft　GS　Wavetable　SW　Synthで再生確認しています





☆起動方法

wwwブラウザをご使用の場合は、CDのルートフォルダにあるindex.htmを開いてください。

画像を直接ご覧になる場合にはimgフォルダに画像が収録されています。
音楽を直接お聞きになる場合にはmidiフォルダに音楽（MIDIファイル）が収録されています。
(詳細は次の「収録内容」を参照してください。)
